path()

path(string  $type, string  $name, string  $granularity = self::GRAN_YEAR) : string

Retourne le chemin vers un fichier de log déterminé à partir des paramètres $type, $name et $granularity.

(ex: /Applications/MAMP/www/monsite/logs/erreurs/201202/201202_erreur_connexion.log) Elle créé le chemin si il n'éxiste pas.
